RENOMÉ Beauty — GitHub Pages ready

Что внутри:
- index.html — главная страница, все фото мастеров встроены внутрь файла. Папка assets не нужна.
- menu.html — меню/кофе/маркет блок.
- marketplace.html — витрина/магазин.
- .nojekyll — служебный файл для GitHub Pages.

Как загрузить на GitHub:
1. Распакуй этот ZIP.
2. Открой репозиторий на GitHub.
3. Нажми Add file → Upload files.
4. Перетащи ВСЕ файлы из распакованной папки, не сам ZIP.
5. Нажми Commit changes.
6. Settings → Pages → Deploy from branch → main → /root → Save.

Важно:
Не нужно создавать папку assets. Сайт работает только с этими файлами.
